# VoxTask

회의 중 말한 내용을 자동으로 분석해 **Action Item(할 일)** 을 추출해주는 Chrome 익스텐션입니다.

![시연 영상](https://github.com/user-attachments/assets/721db4a0-1e7c-4211-bf6d-e42540fde3e1)

---

## 이런 분께 추천해요

- 회의 후 할 일 정리를 매번 손으로 적는 게 번거로운 분
- 화상 회의 중 메모하느라 대화에 집중하기 어려운 분
- 회의 내용을 빠짐없이 캡처하고 싶은 분

---

## 시작 전 준비물

**Groq API 키** — 무료로 발급받을 수 있습니다.

1. [console.groq.com](https://console.groq.com) 접속 후 회원가입
2. **API Keys** 메뉴에서 키 생성
3. 생성된 키(`gsk_...`)를 복사해두기

---

## 설치 방법

1. [Chrome 웹 스토어](https://chrome.google.com/webstore)에서 **VoxTask** 검색
2. **Chrome에 추가** 클릭
3. 팝업에서 **확장 프로그램 추가** 확인

---

## 초기 설정

설치 직후 설정 페이지가 자동으로 열립니다.

| 항목 | 설명 |
|------|------|
| Groq API 키 | 준비한 키를 붙여넣고 저장 |
| 마이크 권한 | **권한 허용** 버튼 클릭 후 브라우저 팝업에서 허용 |

저장하면 바로 사이드 패널에서 녹음을 시작할 수 있습니다.

---

## 사용 방법

1. Chrome 우측 상단의 **VoxTask 아이콘** 클릭 → 사이드 패널 열기
2. **녹음 시작** 버튼 클릭
3. 회의를 진행하면 5초마다 자동으로 분석
4. Action Item이 감지되면 카드로 즉시 표시

> 사이드 패널은 다른 탭을 사용하는 중에도 계속 열려 있습니다.

---

## 자주 묻는 질문

**Q. API 키가 외부에 유출되지 않나요?**
API 키는 내 브라우저(`chrome.storage`)에만 저장되며, 외부 서버로 전송되지 않습니다.

**Q. 인터넷 연결이 필요한가요?**
음성 분석 시 Groq 서버와 통신하므로 인터넷 연결이 필요합니다.

**Q. 어떤 언어를 지원하나요?**
Whisper 모델 기반으로 한국어를 포함한 다국어를 지원합니다.

**Q. 녹음 내용이 저장되나요?**
녹음 원본은 저장되지 않으며, 추출된 Action Item만 세션 동안 표시됩니다.
